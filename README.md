### TextConsoleRPG란?
  
  몬스터와 자동으로 전투하고 성장하여 최종보스를 물리치러 가는 텍스트 기반 콘솔 RPG 게임입니다.

### 주요 콘텐츠
- **전투**
  
  몬스터와 한 턴씩 공격을 주고 받는 자동 전투   
  전투 도중 랜덤으로 스킬이나 아이템을 사용
  
- **승리**

  전투에서 승리시 경험치와 골드 획득 및 레벨업
  랜덤으로 아이템 획득
  
- **상점**

  상점에서 아이템 구매 및 판매 가능
  
- **보스**

  레벨 10 도달시 보스전 입장   
  전투 도중 특정 기믹 수행 필요   
  보스 처치시 엔딩
  
- **기타**

  플레이어 스탯, 인벤토리 확인   
  몬스터 처치 로그 확인
